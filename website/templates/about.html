About this website

This website was built with Flask and deployed using pythonanywhere.com.

How the maps are generated

The maps are generated in Python using a package called Folium. Census data and health inspection data is combined to create an 'Opportunity Metric' which signifies the level of opportunity to open a new restaurant in that zip code. This metric has a theoretical range of 0-100.

How the Opportunity Metric is calculated

The Opportunity Metric has 5 components, equally weighted. These are:

  • Population (higher is better)
  • Median income (higher is better)
  • Restaurants per capita (lower is better)
  • Proportion of selected cuisine (lower is better)
  • Proportion of selected age group (higher is better)
Each component is scaled with a MinMax scaler, then summed. The sum is then multiplied by 20 to give a final range of 0-100.
